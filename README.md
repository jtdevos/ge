# Galactic Empire

It is 3250 in the standard year, 975 years since man has developed
inter-planetary space navigation. Galactic Empire (GE) is a space
exploration and conquest module for Worldgroup BBS systems written in C.

This repository contains the complete original source (`mbmgemp/` and
friends, © 1988–1994 Michael B. Murdock) and a modern remake in
progress: a mechanics specification extracted from the original
(`salvage/`) and a Python simulation core (`ge/`).
